# kredit-data

Публичные данные для приложения [kredit-android](https://github.com/serg-rgb/kredit-android):
актуальные процентные ставки и нормативные параметры ипотечного/кредитного законодательства РФ.

## Зачем отдельный репозиторий

Приложение раз в неделю фетчит эти JSON-файлы, чтобы оставаться актуальным
без нового релиза в RuStore. Конкуренты-калькуляторы статичные — у них в
коде зашиты ставки 2022 года. У нас плашка «Данные актуальны на ДД.ММ.ГГГГ»
показывается в каждом расчёте.

Хранится отдельно потому что:
- Можно править данные, не пуская новый коммит в код приложения.
- GitHub Actions обновляет автоматически с cbr.ru.
- Публичный доступ через `raw.githubusercontent.com` без авторизации.

## Структура

```
data/
├── rates.json       # ставки и макроэкономика (обновляется еженедельно автоматически)
└── legal.json       # нормативная база, лимиты вычетов, льготные программы (обновляется руками)

scripts/
└── update_rates.py  # парсер cbr.ru → rates.json

.github/workflows/
└── update-rates.yml # cron, вызывает update_rates.py каждый понедельник 09:00 МСК
```

## URL для приложения

```
https://raw.githubusercontent.com/serg-rgb/kredit-data/main/data/rates.json
https://raw.githubusercontent.com/serg-rgb/kredit-data/main/data/legal.json
```

Никаких ключей или авторизации не требуется.

## Что обновляется автоматически

| Поле | Источник |
|---|---|
| `cb.keyRate` | [cbr.ru/hd_base/keyrate](https://www.cbr.ru/hd_base/keyrate/) |
| `deposits.averageTop10MaxRate` | [cbr.ru/statistics/avgprocstav](https://www.cbr.ru/statistics/avgprocstav/) |

## Что обновляется вручную

`legal.json` — раз в несколько месяцев или при изменении законодательства:
- Лимиты вычетов (если законодатель поднимет 2 млн → 3 млн)
- Прогрессивная шкала НДФЛ
- Размер маткапитала при индексации (1 февраля)
- Условия льготных программ (продление, изменение ставки, лимита)

## Источники нормативной базы

- [ст. 220 НК РФ — имущественный вычет](https://www.consultant.ru/document/cons_doc_LAW_28165/62f621e5835790398a88f80270fe2cf0b3710b3c/)
- [ст. 214.2 НК РФ — налог на доходы по вкладам](https://www.kommersant.ru/doc/8336830)
- [ФЗ-353 «О потребительском кредите»](https://www.consultant.ru/document/cons_doc_LAW_155986/)
  - ст. 11 (досрочное погашение, период охлаждения)
  - ст. 6.1-1 (ипотечные каникулы)
- [ФЗ-102 «Об ипотеке (залоге недвижимости)»](https://www.consultant.ru/document/cons_doc_LAW_19396/) — обязательное страхование
- [ФЗ № 176-ФЗ от 12.07.2024](https://www.consultant.ru/document/cons_doc_LAW_481261/) — прогрессивная шкала НДФЛ
- [СФР: размер маткапитала](https://sfr.gov.ru/grazhdanam/semyam_s_detmi/materinskij_kapital/razmer)

## Дисклеймер

Данные носят информационный характер. Не являются официальной публикацией
нормативных актов. Перед принятием финансовых решений сверьтесь с
актуальными первоисточниками.
